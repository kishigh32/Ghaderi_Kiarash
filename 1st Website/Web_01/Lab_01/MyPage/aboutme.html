Notes

Overall

XHTML and CSS are the code standards that make websites work. You can view the code behind any website by right clicking in the window and selecting "view source". The first tag you create on a website declares the doctype. Everything in a web object is contained within the opening and closing tags. Everything outside the tags belongs to another object. The head of a site includes information for the browser. The information in the head is not seen by the user of the site. The body of the site includes all of the actual site content.

Paragraph

The < p > < /p > tag allows you to tell the browser explicitly which text in a web document you want to put into paragraph form.

Horizontal Rule

< hr/ > Adds a horizontal line that acts as a separator

Strong Tag

< strong > < /strong > puts the enclosed text in bold.

em Tag

< em > < /em > emphasizes the enclosed text by italicizing it.

Font Size

The header is a system of section titles arranged from h1 to h6 in Descending order. < h1 > has the biggest font size, while < h6 > has the smallest.
